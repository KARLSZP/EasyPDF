# EasyPDF - version 1.2

EasyPDF provides several experimental(beta) functions on PDF files.



## Installation

### Linux (Debian)

*   Python 3.8
*   Git

```shell
git clone git@github.com:KARLSZP/EasyPDF.git
cd EasyPDF/Linux
```



### Windows (Win 10+)

*   Python 3.8
*   Git

```shell
git clone git@github.com:KARLSZP/EasyPDF.git
cd EasyPDF\\Win
```



## Requirements

```shell
pip install -r requirements.txt
```



## Usage

### Linux (Debian)

```shell
chmod +x ./main.py
./main.py
```

or

```shell
python main.py
```



### Windows (Win 10+)

```shell
python main.py
```





**Note:**

*   **out/** and **tmp/** will be automatically made in root dir.



## Functions

### One-File wise

*   Merge Pages



*   Scale Page(s)



*   Split/Extract Page(s)



### Multi-Files wise

*   Merge PDFs
